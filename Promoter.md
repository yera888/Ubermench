**What it is:** a DNA “start site” for [[Transcription]]. Every gene that is transcribed has a promoter. So that RNA can be made so this is that start site where RNA polymerase is recruited.  So it tells the cell, start reading here.

**Where is it?**

- Right next to the gene’s start (upstream of the transcription start site).
    

**How big?**

- Usually ~50–200 bp core; sometimes up to ~1,000 bp including nearby elements.
    

**What does it do?**

- Recruits [[RNA polymerase]] + basic factors to begin [[Transcription]].
